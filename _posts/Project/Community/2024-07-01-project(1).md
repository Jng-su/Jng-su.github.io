> ## Create Project

<br>

### Client

- client 폴더 접속

    ```shell
    cd client
    ```

- vite 환경설치

    ```shell
    npm init vite
    ```

    - project name : `.`
    - select a framework : `React`
    - select a variant : `typescript`


- 의존성 설치

    ```shell
    npm install
    ```


- 설치 확인

    ```shell
    npm run dev
    ```

<br>

### Server


- server 폴더 접속

    ```shell
    cd server
    ```

- 가상환경 생성

    ```shell
    python -m venv ./venv
    ```


- 가상환경 접속

    ```shell
    source ./venv/Scripts/activate
    ```


- FastAPI, uvicorn

    ```shell
    pip insatll fastapi "uvicorn[standard]"
    ```


- SqlModel 설치

    ```shell
    pip install sqlmodel
    ```
