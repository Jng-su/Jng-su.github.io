> ## Create BBS module

<br>

```shell
nest g module boards
```

| Command  | Description                     |
| -------- | ------------------------------- |
| `nest`   | using nest's cli                |
| `g`      | generate                        |
| `module` | schematic that I want to create |
| `boards` | name of the schematic           |
